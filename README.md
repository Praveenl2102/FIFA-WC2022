# FIFA-WC2022
# ⚽ FIFA World Cup 2022 Dashboard | Power BI

## 📌 Overview

The FIFA World Cup 2022 Dashboard is an interactive Power BI project designed to analyze team performances, player statistics, match outcomes, and tournament trends. This dashboard transforms raw FIFA World Cup data into meaningful insights through dynamic visualizations and KPIs, enabling users to explore the tournament from multiple perspectives.

---

## 🎯 Project Objectives

* Analyze FIFA World Cup 2022 match and team performance.
* Compare teams based on wins, goals, and overall tournament progress.
* Identify top-performing players and teams.
* Visualize tournament statistics through interactive dashboards.
* Provide actionable insights using data storytelling techniques.

---

## 📊 Dashboard Features

### 🏆 Tournament Summary

* Total Matches Played
* Total Goals Scored
* Participating Teams
* Tournament Winner
* Average Goals per Match

### ⚽ Team Analysis

* Team Performance Comparison
* Wins, Draws, and Losses
* Goals Scored and Conceded
* Team Ranking Analysis
* Win Percentage Metrics

### 🌟 Player Performance

* Top Goal Scorers
* Top Assist Providers
* Best Performing Players
* Player Comparison Analysis

### 📈 Match Insights

* Goals by Tournament Stage
* Match Outcome Distribution
* Group Stage Performance
* Knockout Stage Analysis

### 🎛 Interactive Features

* Team Filters
* Player Filters
* Tournament Stage Filters
* Dynamic Visualizations
